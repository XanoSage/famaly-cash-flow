# Budgets Notifications Recommendations

Документ фиксирует решения по бюджетам, уведомлениям и рекомендациям.

## First Budget

Первый бюджет строится так:

- сервис анализирует первую выписку;
- предлагает важные категории для лимитов;
- пользователь редактирует лимиты;
- бюджет помечается как предварительный.

## Budget Scope

В MVP нужны:

- общий месячный семейный бюджет;
- лимиты по важным категориям.

Категории для лимитов предлагает сервис на основе первой выписки.

## Budget Warnings

Предупреждения настраиваемые.

Значения по умолчанию:

- 80% лимита: предупреждение;
- 100% лимита: превышение.

## Dashboard Budget Block

На дашборде показываются:

- общий прогресс бюджета;
- категории с риском превышения;
- переход в экран бюджетов.

## In-App Notifications

Уведомления внутри приложения:

- операции, требующие проверки после импорта;
- предупреждения бюджета;
- превышения бюджета;
- напоминания разобрать операции без категории;
- напоминания по наличным.

Telegram-уведомления добавляются позже.

## Insights Frequency

Инсайты показываются:

- на дашборде всегда;
- месячный отчет добавляется позже.

## Recommendation Tone

Тон рекомендаций:

- дружелюбный;
- короткий;
- без давления;
- без чувства вины.

## Period Comparisons

Первое сравнение периодов:

- месяц к месяцу по категориям.

Позже:

- последние 30 дней vs предыдущие 30 дней.
