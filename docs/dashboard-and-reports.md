# Dashboard And Reports

Документ фиксирует решения по дашборду, графикам и отчетам.

## Default Period

Период дашборда настраиваемый. Значение по умолчанию: текущий месяц.

## Top KPI Cards

Верхний блок дашборда показывает:

- расходы;
- доходы;
- накопления;
- использованный бюджет;
- операции без категории;
- рабочие/FOP операции отдельной карточкой.

## Main Expense Chart

Главный график расходов:

- переключатель день/неделя/месяц;
- сравнение с прошлым периодом добавляется позже.

## Category Analytics

Расходы по категориям показываются через:

- bar chart по категориям;
- таблицу деталей.

Подкатегории:

- drill-down из категории в подкатегории;
- таблица деталей рядом или ниже.

## Top Merchants

Top merchants показываются так:

- вкладка `по сумме`;
- вкладка `по количеству операций`;
- фильтр по категории.

## Savings Block

Блок накоплений включает:

- KPI `Накоплено за период`;
- график накоплений;
- сравнение расходы vs накопления;
- прогноз за год при текущем темпе.

## Uncategorized Block

Блок `Разобрать позже` показывается только если есть операции без категории.

Он содержит:

- количество операций;
- сумму;
- кнопку перехода к списку операций.

## Work/FOP Block

Рабочие/FOP операции:

- не входят в семейную аналитику по умолчанию;
- показываются отдельной карточкой;
- по клику открывается отдельный блок или фильтрованный список.

## Reports And Exports

Для MVP:

- CSV export операций.

Позже:

- PDF месячного отчета;
- XLSX отчета.
