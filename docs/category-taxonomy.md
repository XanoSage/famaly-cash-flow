# Category Taxonomy

Документ фиксирует стартовую структуру категорий для MVP. Категории двухуровневые: категория и подкатегория. Третий уровень, split операций и теги добавляются позже.

## Общие правила

- В preview импорта можно быстро добавить новую подкатегорию.
- Банковские категории маппятся на собственную структуру.
- Merchant хранится отдельно от категории и дает детализацию по магазинам, получателям и сервисам.
- Детские расходы выделяются в `Дети`, если расход явно относится к ребенку.
- Рабочие/FOP операции имеют scope `work_fop` и скрываются из семейной аналитики по умолчанию.
- Операции без категории разрешены, но не должны исчезать из аналитики.

## Еда

| Подкатегория | Правило |
| --- | --- |
| Супермаркеты | Обычные продуктовые покупки. Конкретные сети видны через merchant. |
| Рынок | Переводы или покупки у продавцов на рынке. Часто требует ручной проверки. |
| Кофе и перекусы | Маленькие кафе, кофе-точки, перекусы. Определяется по merchant + сумме. |
| Кафе и рестораны | Полноценные кафе, рестораны, фастфуд. |
| Доставка | Готовая еда и доставка еды. Merchant rules могут уточнять доставку продуктов. |

Обычные супермаркеты автоматически идут в `Еда / Супермаркеты`. Крупные чеки супермаркета отправляются на проверку. Лимит настраиваемый, по умолчанию 2500 UAH.

## Дети

| Подкатегория | Правило |
| --- | --- |
| Школа | Регулярная оплата школы и платежи школе как получателю. |
| Спорт | Футбол и другие детские тренировки. Конкретный спорт виден через merchant. |
| Кружки | Внешкольные занятия. |
| Здоровье | Явно детская медицина. |
| Одежда | Детская одежда. |
| Питание в школе | Если операция явно относится к школьному питанию. |
| Развлечения | Детские развлечения. |
| Подарки | Подарки ребенку. |
| Книги и канцтовары | Учебные материалы, книги, канцтовары. |

Футбол сына в MVP идет как `Дети / Спорт`, без третьего уровня `Футбол`. Если понадобится, можно добавить отдельную подкатегорию позже.

## Здоровье

| Подкатегория | Правило |
| --- | --- |
| Аптеки | Аптеки и лекарства для взрослых/семьи. |
| Врачи | Приемы врачей. |
| Анализы | Лаборатории и анализы. |
| Стоматология | Стоматология как отдельная крупная статья. |

Если медицина явно детская, используется `Дети / Здоровье`.

## Дом

| Подкатегория | Правило |
| --- | --- |
| Коммуналка | Коммунальные платежи. |
| Интернет | Домашний интернет. |
| Ремонт | Ремонтные работы и материалы. |
| Товары для дома | Предметы и мелочи для дома. |
| Бытовая химия | Бытовая химия, если операция очевидная или крупная. |
| Вода | Регулярная питьевая вода, например `Кодацька Вода`. |
| Техника и электроника | Техника и электроника для дома. |

Покупки бытовой химии внутри смешанного супермаркетного чека не split-ятся в MVP.

## Услуги

| Подкатегория | Правило |
| --- | --- |
| Доставка и почта | NovaPay, Новая почта и похожие семейные доставки. |
| Банковские комиссии | Комиссии банка, если они есть отдельными операциями. |
| Бытовые услуги | Услуги для дома и семьи. |
| Онлайн-сервисы | Семейные онлайн-сервисы, не рабочие и не подписки. |
| Другое | Прочие услуги. |

Рабочие услуги не смешиваются с семейными: они идут в `Рабочее/FOP` или scope `work_fop`.

## Рабочее/FOP

| Подкатегория | Правило |
| --- | --- |
| Сервисы | Рабочие онлайн-сервисы. |
| Софт | Рабочее ПО. |
| Маркетплейсы | Рабочие маркетплейсы и платформы. |
| Комиссии | Рабочие комиссии. |
| Образование | Рабочее обучение. |
| Другое | Прочие рабочие расходы. |

Потенциально рабочие операции, например Upwork, LinkedIn, ChatGPT и похожие сервисы, отправляются на проверку или правило.

## Путешествия

| Подкатегория | Правило |
| --- | --- |
| Транспорт | Дорога, билеты, трансферы в путешествии. |
| Жилье | Отели, апартаменты, жилье. |
| Еда | Еда в путешествии, если пользователь явно хочет выделить поездку. |
| Развлечения | Развлечения во время поездки. |
| Другое | Прочие расходы поездки. |

Категория может быть пустой в MVP, но она нужна для будущих крупных поездок. В MVP особые случаи можно отмечать комментарием, теги добавляются позже.

## Подписки и цифровые

| Подкатегория | Правило |
| --- | --- |
| Сервисы | Цифровые сервисы для семьи. |
| Софт | ПО и приложения. |
| Медиа | Медиа-подписки. |
| Образование | Образовательные сервисы. |
| Другое | Прочие цифровые расходы. |

Рабочие сервисы уводятся в `Рабочее/FOP`. Повторяемость платежей используется для инсайтов о подписках.

## Питомцы

| Подкатегория | Правило |
| --- | --- |
| Питомцы | Расходы на питомцев. |

Детализация добавляется позже при необходимости.

## Накопления

| Подкатегория | Правило |
| --- | --- |
| Скарбничка | Переводы в Скарбничку. |
| Округления | Округления остатка. |
| Процент от доходов | 1% от доходов и похожие накопительные правила. |

Накопления показываются отдельно и не смешиваются с обычными расходами.

## Переводы и долги

| Подкатегория | Правило |
| --- | --- |
| Переводы жене | Деньги жене из семейного потока. Можно уточнить реальную цель. |
| Переводы физлицам | Переводы людям, если цель неизвестна или неважна. |
| Долги | Долги. |
| Возвраты | Возвраты денег, отдельно от доходов. |
| Другое | Прочие переводы. |

Если перевод физлицу на самом деле рынок, услуга или семейная покупка, его можно переназначить в реальную категорию и сохранить правило по получателю.

## Без категории / Другое

| Подкатегория | Правило |
| --- | --- |
| Без категории | Операция временно не разобрана. |
| Другое | Операция разобрана, но не подходит под другие категории. |

Без категории разрешено, но такие операции показываются отдельно и попадают в список "разобрать позже".
